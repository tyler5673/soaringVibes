Privacy Policy

Soaring Vibes Flight Simulator

This is a free, open-source game. We will never sell your data.

1. Data Collection

Soaring Vibes does not collect, store, or transmit any personal information. The game runs entirely in your browser and stores preferences (such as your aircraft name and color choices) locally on your device only.

2. Local Storage

Your aircraft customizations are saved locally using your browser's localStorage. This data never leaves your device and is not sent to any server.

3. Multiplayer

When playing online, only your aircraft name and position are transmitted to other players via WebSocket connection. No other personal data is shared. You can play offline at any time by disconnecting from multiplayer.

4. Third-Party Services

Soaring Vibes uses the following third-party services:

  • Three.js (CDN) - Rendering library loaded from public CDN
  • WebSocket Server - For optional multiplayer functionality only

5. Cookies

We do not use cookies. No tracking, analytics, or advertising scripts are embedded in this website.

6. Children's Privacy

This game is suitable for all ages. We do not knowingly collect any information from children.

7. Changes to This Policy

If we ever make changes to this policy, we will update this page. Since this is a simple static site with no backend database, any changes would be minor and for transparency purposes only.

8. Contact

If you have any questions about this privacy policy, please contact us through the project's GitHub repository.
